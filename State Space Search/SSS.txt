Brute force need candidate solution set and test candidate

SSS : จากความรู้ที่ทำมากก่อนหน้า จะทำให้ลดขนาด choice ได้
State Space Search 
    The next level brute force.
    Recursion tree ใช้พิจารณาลำดับของ candidate
    partial solution คำตอบก่อน candidate : ใช่ตัด choice
    initial state(start point) -> partial > candidate solution

8-Queen Problem 
    วาง Queen ให้กินกันไม่ได้ในตาราง ;
    sss_v0.1 : มี array 64 ช่อง เลือก มี ไม่มี -> ตาย 2^(8*8);
    sss_v0.2 : ทำคล้ายเดิมแต่นับจำนวนถ้าครบก็พอ
    sss_v0.3 : จะวาง Queen ไว้ 8 แถว [1,6,1,7,5,6,7,8] บอก collumn
    sss_v0.4 : คล้าย 0.3 แต่ห้ามเดิมเลขซ้ำ

DFS and BFS : same as graph
    DFS : stack, less space, loop forever
    BFS : queue, large space, solution near root

Triple and Half Problem
    คำนวน *3 หรือ /2 ให้เท่ากับ input [T,H]
    ทุกปมเป็น candidate 
    0.1 : DFS
    0.2 : สักอย่าง
    BFS : good ไล่เป็นระดับ

Backtracking
    ดู partial state ถ้าไม่ใช่ไม่ต้องทำต่อ 
    permutation by combination

Sum of Subset
    DFS : ดูผลรวมแล้วตอบ
    Backtracking : ถ้าปมไหนบวกเกินก็ไม่ทำต่อ

    constraint and objective function
    Bounding Heuristic เดาให้ดีเกินจริง ไม่แย่เกินจริง
        ไม่ underestimate
